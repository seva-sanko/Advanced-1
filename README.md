# Advanced-1
Серия программ предназначенная для углубленного изучения C++.

В данной программе затрагиваются такие темы, как классы, операции, вложенность. Если говорить подробнее, то реализовано переопределение операций, определение операций с помощью friend-функций, статические данные и функции и абстрактные классы.
